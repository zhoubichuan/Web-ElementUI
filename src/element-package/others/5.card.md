# 五.Card

::: demo

```vue
<template>
  <web-card class="box-card">
    <div slot="header" class="clearfix">
      <span>卡片名称</span>
      <web-button style="float: right; padding: 3px 0" type="text"
        >操作按钮</web-button
      >
    </div>
    <div v-for="o in 4" :key="o" class="text item">
      {{ "列表内容 " + o }}
    </div>
  </web-card>
  <style>
    .text {
      font-size: 14px;
    }

    .item {
      margin-bottom: 18px;
    }

    .clearfix:before,
    .clearfix:after {
      display: table;
      content: "";
    }
    .clearfix:after {
      clear: both;
    }

    .box-card {
      width: 480px;
    }
  </style>
</template>
```

:::
